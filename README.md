# 🏡 BlueBrick Real Estate

BlueBrick Real Estate is a modern, responsive frontend web application designed to showcase premium residential and commercial properties. The project focuses on delivering a clean user experience through responsive layouts, interactive UI components, smooth animations, and a functional contact form powered by EmailJS.

## 🌐 Live Demo

🔗 https://fantastic-tiramisu-3d75d7.netlify.app/

---

## ✨ Features

- Responsive and modern UI
- Animated hero section
- Property categories (Sale, Rent & Off-Plan)
- Dynamic property pages
- Developer showcase pages
- Contact form powered by EmailJS
- FAQ accordion
- Mobile-friendly navigation
- Smooth animations and transitions

## 🚀 Getting Started

1. Clone or download the repository.
2. Open the project in VS Code.
3. Configure your EmailJS credentials (Public Key, Service ID, and Template ID).
4. Run the project using **Live Server** or open `index.html` in your browser.
